# Agent Instructions

Use the local skill kit:

```text
./agent-skill-kit
```

Before substantial work:

1. Read `./agent-skill-kit/skill-library-map.md`.
2. Choose the narrowest relevant skill from `./agent-skill-kit/skills`.
3. Follow the workflow and references inside that skill.

For new features, start with `spec-driven-development`.

For PR-level review, start with `code-review`.

For sensitive data, permissions, external tools, production actions, or MCP/tool access, include `agent-security-review`.
