# Project Start Example

Use this prompt when starting a new project with the kit:

```text
This project uses agent-skill-kit.

Before making changes:
1. Read AGENTS.md.
2. Read agent-skill-kit/skill-library-map.md.
3. Select the narrowest relevant skill from agent-skill-kit/skills.
4. If the task is unclear, start with spec-driven-development and produce a spec before implementation.
5. Before merge-level review, use code-review.
```

Example feature prompt:

```text
Use spec-driven-development to create a feature spec for user profile editing. Include acceptance criteria, BDD scenarios, API contract, data model notes, security notes, and testing plan. Do not implement yet.
```
