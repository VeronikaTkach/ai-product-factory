# Commerce Policy: Office Supply Reorder Agent

## Allowed Actions

- Discover: allowed.
- Draft order: allowed.
- Pay: approval-required.
- Refund: draft-only.

## Spending Limits

- Per transaction: 150 USD.
- Daily: 300 USD.
- Monthly: 1000 USD.

## Merchant Rules

- Allowed merchants: approved office supply vendors.
- Blocked merchants: unknown marketplaces.

## Consent Requirements

- User must approve merchant, items, quantity, final amount, and delivery date.
- Vibe Diff required before payment.

## Audit Trail

- Receipt storage: finance folder or expense system.
- Approval record: required.
- Trace retention: keep all paid transactions.
